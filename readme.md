# 📊 Student Performance Analysis

## 📌 Project Overview
This project analyzes student performance data using Python and Tableau.  
The goal of this project is to understand student performance patterns, generate insights, and create dashboards for decision-making.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Tableau
- Jupyter Notebook

---

## 📂 Dataset
Dataset used:
Student Performance Dataset from Kaggle

The dataset contains:
- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## 📊 Steps Performed

### 1️⃣ Data Collection
- Collected dataset from Kaggle
- Imported dataset into Jupyter Notebook

### 2️⃣ Data Understanding
- Checked rows and columns
- Explored dataset structure
- Understood features and datatypes

### 3️⃣ Data Cleaning
- Checked missing values
- Checked duplicate values
- Cleaned messy data
- Standardized column names

### 4️⃣ Exploratory Data Analysis (EDA)
- Categorical Analysis
- Numerical Analysis
- Comparison Analysis
- Correlation Analysis

### 5️⃣ Data Visualization
Created:
- Bar Charts
- Pie Charts
- Histograms
- Boxplots
- Heatmaps

### 6️⃣ Tableau Dashboard
Created an interactive dashboard containing:
- KPI Card
- Gender Distribution
- Average Math Score
- Comparison Charts

---

## 📈 Key Insights
- Female students performed better in reading and writing.
- Male students scored slightly higher in math.
- Students who completed test preparation performed better overall.
- Reading and writing scores showed strong positive correlation.

---

## 🏫 Business Recommendations
- Schools should encourage preparation programs.
- Additional support should be provided to weaker students.
- Student performance should be monitored regularly.
- Schools can improve performance using data-driven strategies.
